:-Fontawesome-:

  • First you have to Install fontawesomefree module in your system.

    Syntax-: pip install fontawesomefree

  • You have to register fontawesomefree module in INSTALLED_APPS

    INSTALLED_APPS = [ 'fontawesomefree' ]

  • After that you have to link this file into your html file within head tag

    < link href="{% static 'fontawesomefree/css/all.min.css' %}" rel="stylesheet" type="text/css">

  • Syntax of using font icon in html body is given below

    <i class="fa fa-home">Home < / i>
